C++ Шпаргалка Типы данных      С++ язык со строгой типизацией. Каждый объект обязан иметь определенный тип. И этот тип определяет, какие значения может принимать объект, какие операции с ним можно производить и сколько байт в памяти он будет занимать.  Вы можете удивиться, но размер переменной с любым типом данных зависит от компилятора и/или архитектуры компьютера. Язык C++ гарантирует только их минимальный размер.     В языке C++ определены следующие базовые типы данных: Тип Минимальный размер Диапазон значений Описание void не имеет размера нет Представляет неопределенный тип. Чаще всего используется для функций которые не должны возвращать никакого значения. bool 1 байт (0 или false) и (1 или true) Логический тип. Может принимать одну из двух значений true (истина) и false (ложь) short 2 байта   Представляет целое число.  int 2 байта от −2 147 483 648 до 2 147 483 647 Представляет целое число.   long 4 байта   Представляет целое число.  long long 8 байт   Представляет целое число.  float 4 байта   Представляет вещественное число с плавающей точкой double 8 байт   Представляет вещественное число с плавающей точкой long double 8 байт   Представляет вещественное число с плавающей точкой char 1 байт   Представляет код символа в таблице ASCII char16_t 2 байта     char32_t 4 байта     wchar_t 1 байт       Если хотите узнать размер в байтах для типов данных на вашем компьютере используйте оператор sizeof. Следующая программа выводит список типов и их размеры в байтах   #include   int main() {     std::cout << "bool:\t\t" << sizeof(bool) << " bytes" << std::endl;     std::cout << "char:\t\t" << sizeof(char) << " bytes" << std::endl;     std::cout << "wchar_t:\t" << sizeof(wchar_t) << " bytes" << std::endl;     std::cout << "char16_t:\t" << sizeof(char16_t) << " bytes" << std::endl;       std::cout << "char32_t:\t" << sizeof(char32_t) << " bytes" << std::endl;         std::cout << "short:\t\t" << sizeof(short) << " bytes" << std::endl;     std::cout << "int:\t\t" << sizeof(int) << " bytes" << std::endl;     std::cout << "long:\t\t" << sizeof(long) << " bytes" << std::endl;     std::cout << "long long:\t" << sizeof(long long) << " bytes" << std::endl;     std::cout << "float:\t\t" << sizeof(float) << " bytes" << std::endl;     std::cout << "double:\t\t" << sizeof(double) << " bytes" << std::endl;     std::cout << "long double:\t" << sizeof(long double) << " bytes" << std::endl;     return 0; } Copyright © 2021 DAYWER.RU.
