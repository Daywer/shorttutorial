C++ Шпаргалка Переменные Переменная - это «ячейка» оперативной памяти компьютера, в которой может храниться какая-либо информация. До начала использования, переменную нужно объявить. Делается это таким образом. Сначала указываем тип, затем имя переменной. Например:      int x;        // создает переменную целого типа с именем "x"      int x{5};   // создает переменную целого типа с именем "x" и присваивает ей значение 5     int x=5;    //  тоже самое что и выше, другая форма записи. ВНИМАНИЕ!     - Тип у переменной указывается только один раз, в момент объявления.     - Внутри программы значение переменной можно перезаписывать таким образом          x=22;    // так можно          x{22};   // так нельзя. такая форма записи допускается только при объявлении переменной.   Примеры объявлений переменных разного типа: bool x=true;  // обявлена переменная логического типа с именем "x" и значением true (ИСТИНА). bool x=0;       // обявлена переменная логического типа с именем "x" и значением 0 (ЛОЖЬ). int a=5;         // обявлена переменная целого типа с именем "a" и значением 55.
