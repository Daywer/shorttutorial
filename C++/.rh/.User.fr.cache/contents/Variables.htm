C++ Шпаргалка Переменные Переменная - это «ячейка» оперативной памяти компьютера, в которой может храниться какая-либо информация. До начала использования, переменную нужно объявить. Делается это таким образом. Сначала указываем тип, затем имя переменной. Например:      int x;        // создает переменную целого типа с именем "x"      int x{5};   // создает переменную целого типа с именем "x" и присваивает ей значение 5     int x=5;    //  тоже самое что и выше, другая форма записи. ВНИМАНИЕ!     - Тип у переменной указывается только один раз, в момент объявления.     - Внутри программы значение переменной можно перезаписывать таким образом          x=22;    // так можно          x{22};   // так нельзя. такая форма записи допускается только при объявлении переменной.   Примеры объявлений переменных разного типа: bool x=true;                // объявлена переменная логического типа с именем "x" и значением true (ИСТИНА). bool x=0;                     // объявлена переменная логического типа с именем "x" и значением 0 (ЛОЖЬ). int a=5;                        // объявлена переменная целого типа с именем "a" и значением 55. char symbol=84;        // объявлена переменная символьного типа с именем "symbol" и значением 84 ( символ "T" из ASCII таблицы). string name = "John Smith"; // объявлена переменная типа "строка" с именем name и значением John Smith.                                                     // значения строковых переменных необходимо записывать внутри двойных кавычек.
