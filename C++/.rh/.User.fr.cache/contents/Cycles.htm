C++ Шпаргалка Циклы Циклы служат для повторения участка кода. Цикл FOR продолжает выполняться пока условие выполнения цикла возвращает ИСТИНУ. for ( выражение инициализации; выражение условия выполнения цикла;  выражение выполняющееся в конце каждого цикла)  {     // Между фигурными скобками находится тело цикла.    //  То есть сюда помещают программный код    //  Если строка кода всего одна, то фигурные скобки не нужны. }   Ниже показаны примеры использования циклов. Цикл FOR следует понимать так. int i=0 - перед началом выполнения объявить переменную типа "int" с именем "i" и присвоить ей значение ноль. i<100 - продолжать цикл пока значение переменной "i" меньше ста. i++ - в конце каждого цикла выполнять инкремент переменной "i", то есть увеличивать значение на единицу. Переменную "i" в данном случае принято называть счетчиком цикла или итератором.   Цикл FOR        for (int i=1; i<100; i++)             // одна строка кода;  Или так:           for (int i=1; i<100; i++)             {                 // одна или несколько строк кода;             } ВНИМАНИЕ! 1. Инициализировать можно сразу несколько переменных, НО ТОЛЬКО ОДНОГО ТИПА ДАННЫХ. 2. Условие выполнения может быть усложнено логическими операторами И, ИЛИ, НЕ. 3. Выражений выполняющихся в конце цикла может быть несколько.   Пример: for (int i=1, a=2 ; i<10 && a>10 ; i++, a--)             {                 // одна или несколько строк кода;             }     Цикл WHILE выполняется до тех пор пока выражение в скобках возвращает ИСТИНУ. В скобках может быть указано любое выражение сравнения, либо true или false. Условие выполнения может быть усложнено логическими операторами И, ИЛИ, НЕ. Цикл WHILE        while (true)                 // одна строка кода;  Или так:            while (true)           {                 // одна или несколько строк кода;           } Цикл DO/WHILE В цикле DO/WHILE условие ставится в конце, поэтому не смотря на условие цикл гарантированно будет выполнен хотя бы один раз. В скобках может быть указано любое выражение сравнения, либо true или false. Условие выполнения может быть усложнено логическими операторами И, ИЛИ, НЕ.        do                  // одна строка кода;                     while ( true ) ;    Или так:       do {                  // одна или несколько строк кода;                         }  while ( true ) ; Copyright © 2021 DAYWER.RU.
