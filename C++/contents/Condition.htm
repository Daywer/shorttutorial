Условный оператор IF/ELSE

 

Если условие указанное в круглых скобках выполняется, то есть результат возвращает ИСТИНУ, тогда выполняется  одна строка программы.

Выражение условия может быть усложнено логическими операторами И, ИЛИ, НЕ.

     

       if ( n <  100)   // одна cтрока кода ;      // если i меньше 100 выполняем какую-то строку программы

 Или так:

       if ( n < 100) 

                          // одна строка кода  ;                    // если i меньше 100 выполняем какую-то строку программы

                  else

                          // другая строка кода ;                // иначе выполняем другую строку программы

Если нужно выполнить несколько строк программы, то они должны быть окружены фигурными скобками { }.

 

      if ( n == 2 )    // если n равно двум

                   { 

                          // одна или несколько строк кода;

                   }

 Или так:

     if ( n != 2 )  // если n не равно двум

                   { 

                         // одна или несколько строк кода;

                   }

           esle

                   { 

                         // одна или несколько строк кода;

                   }
