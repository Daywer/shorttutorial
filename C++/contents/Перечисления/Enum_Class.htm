Перечисления с областью видимости

Перед этим материалом, изучите  "Перечислители" >>  Enum

 

Данное перечисление отличается тем, что перечислители с одинаковыми именами теперь можно использовать в разных перечислениях, но при обращении 

нужно указывать области видимости

Пример:

enum class Colors1

{

YELLOW,

ORANGE,

BLUE

};

enum class Colors2

{

RED,

GREEN,

BLUE

};

Colors2 mycolors = Colors2::Blue;

 

Мы видим что Blue встречается в перечислениях Colors1 и Colors2. Поэтому  запись "Colors2::Blue" указывает на то 

что используется перечисление Blue из Colors2

!ВНИМАНИЕ:

  1.  У нас появилось ключевое слово class.

  2.  Мы присвоили объекту mycolors, имеющиему тип данных Colors2, значение Blue из области видимости Colors2. 

  3.  Мы использовали оператор расширения области видимости ::
