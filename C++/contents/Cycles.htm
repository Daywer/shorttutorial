C++ Шпаргалка

Циклы

 

Цикл FOR 

      for (int i=1; i<100; i++)

            // одна строка кода;

 Или так:    

      for (int i=1; i<100; i++)

            {

                // одна или несколько строк кода;

            }

 

Цикл WHILE

       while (true)

                // одна строка кода;

 Или так:    

       while (true)

          {

                // одна или несколько строк кода;

          }

Цикл DO/WHILE

В цикле DO/WHILE условие ставится в конце, поэтому не смотря на условие цикл гарантированно будет выполнен хотя бы один раз.

       do 

                // одна строка кода;             

       while ( true ) ;

 

 Или так: 

     do {

                 // одна или несколько строк кода;           

             }  while ( true ) ;
