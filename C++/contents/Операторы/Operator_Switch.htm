Оператор переключения Switch

Выполняет участок кода при котором значение переменной-аргумента равно метке.

int x = 2;

switch ( x )        // где "x" переменная целого типа

{

    case  1:         //  если х == 1 выполняется одна или несколько строк кода

                          break;

    case  2:         //  если х == 2 выполняется одна или несколько строк кода

                          break;

    case  3:         //  если х == 3 выполняется одна или несколько строк кода

                          break;

    case  4:         //  если х == 4 выполняется одна или несколько строк кода

                          break;

    default:   ;        //  если x не равно ни одному из выше перечисленного

}

Агрументом оператора SWITCH, могут быть данные типов: int, char, пользовательские типы образованные от enum, enum class

 

 

У оператора  Switch есть известные особенности работы. Рекомендуем ознакомится. (см. SWITCH/BREAK)
