Присваивание

     int a = 4;    // переменной "а" присваивается значение 4

     std::cout << a; 

результатом будет вывод на консоль числа 4

 

Сложение

     int a = 4, b = 5;    // переменной "а"  присваивается значение 4, переменной "b" значение 5

     std::cout << a + b; 

результатом будет вывод на консоль суммы чисел 4 и 5, то есть 9.

 

Вычитание

     int a = 4, b = 5;    // переменной "а" присваивается значение 4, переменной "b" значение 5

     std::cout << b-a;  // операция вычитания 5-4

результатом будет вывод на консоль разности чисел 5 и 4, то есть 1.

 

Унарный плюс

     int a = +5   // переменной "а" присваивается значение 5

     std::cout << a;  

результатом будет вывод на консоль числа 5. Как и в математике унарный плюс опускается. Число без знака и так является положительным

 

Унарный минус

     int a = -5   // переменной "а"присваивается значение минус 5

     std::cout << a;  

результатом будет вывод на консоль числа -5. Унарный минус делает число отрицательным.

 

Умножение

     int a = 3, b = 2;   // переменной "а" присваивается значение 3, переменной "b" значение 2

     std::cout << a * b;  

результатом будет вывод на консоль произведения чисел 3 и 2, то есть 6.

 

Деление 

Для вещественных чисел (данных типа float, double, long double .......)

     float a = 3, b = 2;   // переменной "а" присваивается значение 3, переменной "b" значение 2

     std::cout << a / b;  

результатом будет вывод на консоль частного чисел 3 и 2, то есть 1,5.

Для целый чисел (данных типа int, long int, long long .......)

     int a =3 , b = 2;   // переменной "а" присваивается значение 3, переменной "b" значение 2

     std::cout << a / b;  

результатом будет вывод на консоль частного чисел 3 и 2 без остатка, то есть 1. Так как переменные целого типа данных не могут хранить числа после запятой.

 

Операция модуль (Остаток от деления целых чисел)

     int a = 9, b = 7;   // переменной "а "присваивается значение 9, переменной "b" значение 7

     std::cout << a % b;  

результатом будет вывод на консоль остатка деления, то есть 2.

ВНИМАНИЕ! Данная операция не применима к вещественным числам.

 

Инкремент (операция увеличение любого числа на единицу)

Префиксный инкремент

     int a = 9;     // переменной "а" присваивается значение 9

     std::cout << ++a;  

Так как инкремент префиксный то сначала будет значение переменной "a" увеличено на единицу.

Затем на консоль будет выведено число 10

 

Суффиксный или постфиксный инкремент

     int a = 9;     // переменной а присваивается значение 9

     std::cout << a++ <<endl;   // выведет на консоль 9 и увеличится на единицу

     std::cout << a;                    // выведет число 10.

Так как инкремент пофстфиксный, то сначала на консоль будет выведено число 9.  Затем значение переменной "а"  будет увеличено на единицу.

При повторном выводе будет выведено число 10.

ВНИМАНИЕ! Если операция инкремента применяется к вещественному числу, то увеличиваться будет только целая часть.

 

Декремент (операция уменьшения любого числа на единицу)

Префиксный декремент

     int a = 9;     // переменной "а" присваивается значение 9

     std::cout << --a;  

Так как декремент префиксный то сначала значение переменной "a" будет уменьшено на единицу.

Затем на консоль будет выведено число 8

 

Суффиксный или постфиксный декремент

     int a = 9;     // переменной "а" присваивается значение 9

     std::cout << a-- <<endl;   // выведет на консоль 9 и уменьшится на единицу

     std::cout << a;                    // выведет число 8.

Так как декремент пофстфиксный то сначала на консоль будет выведено число 9. Затем значение переменной "а"  будет уменьшено на единицу.

При повторном выводе будет выведено число 8.

ВНИМАНИЕ! Если операция декремента применяется к вещественному числу, то уменьшатся будет только целая часть.
