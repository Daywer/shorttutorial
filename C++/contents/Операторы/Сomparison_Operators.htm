Операторы сравнения

Операция (выражение) Оператор Синтаксис выражения Перегружаемый
Равенство == a == b Да
Неравенство != a != b Да
Больше > a > b Да
Меньше < a < b Да
Больше или равно >= a >= b Да
Меньше или равно <= a =< b Да

 

Равенство

     int a = 4, b=5;    // переменной "а" присваивается значение 4, переменной "b" значение 5 

     if (a == b) std::cout "TRUE";

                else 

                      std::cout "FALSE";

результатом будет вывод на консоль "FALSE". Так как сравнение двух чисел 4 и 5 вернуло результат ЛОЖЬ. Поскольку утверждение что 4 равно 5 ложно.

 

Неравенство

     int a = 4, b=5;    // переменной "а" присваивается значение 4, переменной "b" значение 5

     if (a != b) std::cout "TRUE";

                else 

                      std::cout "FALSE";

результатом будет вывод на консоль "TRUE". Так как сравнение двух чисел 4 и 5 вернуло результат ИСТИНА. Поскольку утверждение что 4 не равно 5 истинно.

 

Больше

     int a = 4, b=5;    // переменной "а" присваивается значение 4, переменной "b" значение 5

     if (a > b) std::cout "TRUE";

                else 

                      std::cout "FALSE";

результатом будет вывод на консоль "FALSE". Так как сравнение двух чисел 4 и 5 вернуло результат ЛОЖЬ. Поскольку утверждение что 4 больше 5 ложно.

 

Меньше

     int a = 4, b=5;    // переменной "а" присваивается значение 4, переменной "b" значение 5

     if (a < b) std::cout "TRUE";

                else 

                      std::cout "FALSE";

результатом будет вывод на консоль "TRUE". Так как сравнение двух чисел 4 и 5 вернуло результат ИСТИНА. Поскольку утверждение что 4 меньше 5 истинно.

 

Больше или Равно

     int a = 33;  // переменной "а" присваивается значение 33

     if (a >= 10) std::cout "TRUE";

                else 

                      std::cout "FALSE";

результатом будет вывод на консоль "TRUE". Так как выражение истинно когда значение переменной "а" равно 10 или больше него. 

 

Меньше или Равно

     int a = 33;  // переменной "а" присваивается значение 33

     if (a <= 10) std::cout "TRUE";

                else 

                      std::cout "FALSE";

результатом будет вывод на консоль "FALSE". Так как выражение вернет результат ИСТИНА когда значение переменной "а" либо равно 10, либо  меньше него. Во всех остальных случаях выражение вернет результат ЛОЖЬ.
